行情解析器数据回调接口

行情解析器回调接口

定义了行情解析器需要实现的回调接口，用于处理不同类型的行情数据 实现该接口的类可以接收并处理从解析器推送的各种市场数据

定义了接收行情解析器数据的回调方法 任何需要处理行情数据的组件都需要实现该接口
