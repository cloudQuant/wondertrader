差价执行器类

差价执行器继承自ExecuteContext、ITrdNotifySink和IExecCommand接口，负责根据目标仓位和当前仓位的差值执行交易操作 差价执行器通过计算目标仓位与实际仓位的差异，自动生成交易指令，实现仓位管理和调整 同时处理交易通道的推送回调，如成交、委托、资金、持仓等信息
