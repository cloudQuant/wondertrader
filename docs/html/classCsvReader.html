bool CsvReader::load_from_file ( const char *  filename)

从文件加载CSV数据

参数
filename文件路径
返回
加载是否成功

该方法会读取CSV文件的第一行作为字段名，并构建字段名到列索引的映射

参数
filename文件路径
返回
加载是否成功

该方法首先检查文件是否存在，然后打开文件并读取第一行作为字段名。 处理过程包括：

  1. 检测并跳过UTF-8 BOM标记
  2. 清除特殊字符
  3. 将字段名转为小写
  4. 分割并存储字段名到列索引的映射
