线程池核心实现

本文件包含线程池的核心类：pool<Task, SchedulingPolicy>。

线程池是一种在同一进程内实现异步和并行处理的机制。 pool 类提供了一种便捷的方式，将函数对象作为异步任务调度执行。 通过使用自定义的调度器，可以轻松控制这些任务的调度方式。
