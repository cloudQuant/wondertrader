详细描述

循环任务函数对象

此函数对象封装了一个返回布尔值的线程函数对象。 被封装的任务函数通过调用操作符 () 来执行，并且它会在定期的时间间隔内执行， 直到返回 false 为止。间隔长度可以为零。 请注意，只要任务处于循环状态，线程池的线程就会一直被占用。 这种任务类型适用于需要定期执行的操作，如定时检查或监控。
