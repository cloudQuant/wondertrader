高频交易引擎类

高频交易引擎继承自WtEngine基类，负责管理和执行高频交易策略 主要功能包括：

  1. 接收市场数据（Tick、委托队列、委托明细、成交明细等）
  2. 管理高频策略上下文
  3. 处理交易信号和订单
  4. 提供市场数据查询接口 高频交易引擎主要针对低延迟、高频率的交易场景
