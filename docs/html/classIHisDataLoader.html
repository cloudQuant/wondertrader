加载最终历史K线数据

加载经过处理的最终历史K线数据，和loadRawHisBars的区别在于， loadFinalHisBars加载的是经过框架处理好的最终数据（如已经过复权或合成了主力合约）， 系统不再对这些数据进行处理。

参数
obj回传用的上下文指针，原样返回即可
stdCode标准化合约代码
periodK线周期

返回
