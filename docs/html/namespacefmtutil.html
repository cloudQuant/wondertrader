const char* 返回格式化后的字符串指针

使用线程本地静态缓冲区存储格式化后的字符串，并返回其指针 注意返回的指针指向线程本地静态缓冲区，这意味着：

  1. 返回的字符串只在当前线程中有效
  2. 下一次调用会覆盖同一线程中的上一次结果
  3. 如果格式化后的字符串长度超过BUFSIZE，将导致缓冲区溢出
