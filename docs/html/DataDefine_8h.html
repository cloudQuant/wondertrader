详细描述

WonderTrader数据存储系统数据结构定义文件

\project WonderTrader

作者
Wesley

该文件定义了所有与数据存储相关的数据结构，包括：

  1. 数据块类型枚举
  2. 数据块头部结构
  3. 实时数据块定义（K线、Tick、逐笔成交等）
  4. 历史数据块定义 用于支持WonderTrader的各类行情数据的存储与读取

在文件 DataDefine.h 中定义.
