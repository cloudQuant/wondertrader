◆ bind_core()

static bool CpuHelper::bind_core ( uint32_t  i)
inlinestatic

将当前线程绑定到指定的CPU核心

参数
i指定的CPU核心索引，从0开始计数
返回
bool 绑定成功返回true，失败返回false

此函数会尝试将调用线程绑定到指定索引的CPU核心上执行 如果指定的核心索引超出系统可用核心数，将返回false 不同操作系统下使用不同的API实现绑定功能
