行情工厂主模块

行情工厂(QuoteFactory)是一个用于接收、处理和分发行情数据的组件 主要功能包括：

  1. 加载并初始化各种行情解析器
  2. 收集和处理各种市场数据
  3. 通过UDP和共享内存分发行情数据
  4. 支持全天候模式和交易时段模式
