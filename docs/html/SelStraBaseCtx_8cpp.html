inline

生成选股策略上下文ID

返回
返回唯一的上下文ID

通过原子操作生成唯一的上下文ID，起始值为3000 每次调用此函数都会返回一个新的ID值（当前值），并将内部计数器加1 这确保了每个选股策略实例都有唯一的标识符
