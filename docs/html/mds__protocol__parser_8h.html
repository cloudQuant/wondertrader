◆ MdsParser_GetMdStreamIdString()

const char * MdsParser_GetMdStreamIdString ( uint8  exchId,
uint8  bodyType,
uint8  subStreamType 
)

返回字符串形式的行情数据类型

上交所 (C5):

  • MD001 指数行情数据
  • MD002 股票(A、B股)行情数据
  • MD003 债券分销类产品
  • MD004 基金行情数据
  • MD201 债券行情数据
  • M0301 期权交易行情

深交所 (C3):

  • 010 现货(股票,基金,债券等)集中竞价交易快照行情
  • 020 质押式回购交易快照行情
  • 030 债券分销快照行情 (* 属于综合业务，本系统不处理)
  • 040 期权集中竞价交易快照行情 (==> M0301)
  • 060 以收盘价交易的盘后定价交易快照行情 (* 属于综合业务，本系统不处理)
  • 061 以成交量加权平均价交易的盘后定价交易快照行情 (* 属于综合业务，本系统不处理)
  • 900 指数快照行情 (==> MD001)
  • 910 成交量统计指标快照行情
参数
exchId交易所代码
bodyType快照数据对应的消息类型
subStreamType行情数据类别
参见
eMdsSubStreamTypeT
返回
行情数据类型字符串
