WonderTrader数据处理辅助工具头文件

@project WonderTrader

作者
Wesley
日期
2020/03/30

该文件定义了WonderTrader数据处理辅助工具的接口函数， 包括数据转换、数据导出、数据读取和数据重采样等功能。 这些函数主要用于处理K线、Tick、委托明细、委托队列和成交数据， 支持二进制数据与CSV格式之间的转换，以及数据的读取和重采样操作。

对K线数据进行重采样

参数

cbCnt数据计数回调函数
fromTime开始时间

将指定文件中的K线数据进行重采样，生成新的周期的K线数据。 例如将分钟线采样为5分钟线、日线等。重采样的数据通过回调函数返回给调用者。

参数
barFile输入K线数据文件路径
cbK线数据获取回调函数，用于返回重采样后的数据
cbCnt数据计数回调函数
fromTime开始时间
endTime结束时间
period目标K线周期，如"m5"/"d1"
times周期倍数，如将分钟线采样为5分钟线，则为5
sessInfo交易时段信息
cbLogger日志回调函数，默认为NULL
bAlignSec是否按秒对齐，默认为false
返回
WtUInt32 重采样后的数据数量

该函数将指定文件中的K线数据进行重采样，生成新的周期的K线数据。 首先读取原始数据，然后根据指定的周期和倍数进行采样， 最后通过cb回调函数返回采样后的数据。 该函数支持将分钟线采样为更高周期的K线，如将分钟线采样为5分钟线、日线等。
