template<typename Pool>
class boost::threadpool::immediately< Pool >

立即关闭策略，不等待任务或工作线程终止

该关闭策略不等待任何任务完成。它会清空任务队列，并且不等待工作线程终止。 尽管如此，当前正在执行的任务还是会被完全处理完毕。 这是最快速的关闭策略，适用于不需要等待队列中任务完成的场景。
