\project WonderTrader

作者
Wesley
日期
2020/03/30

该文件实现了WonderTrader中的选股策略回测模拟器， 负责模拟选股策略的执行、信号处理、仓位管理以及回测结果的计算与导出。 主要功能包括：

  1. 处理各类事件（K线、Tick、调度等）
  2. 管理策略持仓和信号
  3. 计算动态盈亏
  4. 生成回测报告和输出结果
