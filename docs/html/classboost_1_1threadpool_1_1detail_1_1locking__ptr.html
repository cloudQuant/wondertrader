详细描述

template<typename T, typename Mutex>
class boost::threadpool::detail::locking_ptr< T, Mutex >

带有作用域锁定机制的智能指针

该类是对易变指针的包装器。它通过锁定传入的互斥量实现对内部指针的同步访问。 在构造时自动锁定互斥量，在析构时自动解锁，从而确保线程安全的访问模式。

模板参数
T指针指向的对象类型
Mutex互斥量类型，用于同步访问
