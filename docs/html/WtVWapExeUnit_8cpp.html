◆ calTmStamp()

double calTmStamp ( uint32_t  actiontime)
inline

计算时间戳在交易时间中的分钟位置

将HHMMSS格式的时间转换为交易日内的分钟数，考虑了交易时段和中午休市 主要用于将当前时间映射到VWAP成交量分布数组中的位置 交易时间分为上午和下午两段：9:30-11:30和13:30-15:00

参数
actiontime时间戳，格式为HHMMSS或HHMMSSMMM
返回
交易日内的分钟数，范围0-240，其中0表示开盘前，240表示收盘时
