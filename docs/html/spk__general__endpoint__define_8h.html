连接断开后的回调函数的函数原型定义

回调函数说明:

  • 仅用于通知客户端连接已经断开, 无需做特殊处理, 通信线程会自动尝试重建连接

  • 若回调函数返回小于0的数, 则通信线程将中止运行

线程说明:

返回值

<0小于0, 通信线程将中止运行
