Tick数据对象

Tick数据类

用于存储和管理实时市场数据（Tick数据）的类 内部封装WTSTickStruct结构体，包含了实时行情数据 如最新成交价、委买委卖相关数据、成交量、持仓量等 继承WTSPoolObject类实现对象池管理，提高内存使用效率

封装的主要目的是出于跨语言的考虑，将底层数据结构封装成类，便于对外提供接口 同时提供了丰富的方法来访问其中的各个字段
