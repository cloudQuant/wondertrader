template<typename Task = prio_task_func>
class boost::threadpool::prio_scheduler< Task >

实现优先级排序的调度策略

该容器实现了基于任务优先级的调度策略。 具有最高优先级的任务将首先被移除并执行。 必须能够使用 operator< 比较两个任务。

模板参数
Task实现 operator() 和 operator< 的函数对象。operator< 必须是偏序关系。
