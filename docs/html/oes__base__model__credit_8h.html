注解
记录合约负债变动流水, 包括: 开仓流水、归还流水、以及合约展期流水
  • 对于归还流水, 发生金额、发生费用、发生利息、发生证券数量为负数
  • 对于开仓流水, 对应数据为正数
  • 对于合约展期流水, 发生金额、发生费用等数据为0
