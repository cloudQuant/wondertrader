字符串拷贝函数

重写的字符串拷贝函数，直接使用memcpy实现字符串拷贝功能 最初设计目的是为了提高性能，但最新测试结果显示性能提升不显著

参数
des目标字符串缓冲区
src源字符串
len要拷贝的字符串长度，默认为0，表示自动计算源字符串长度
返回
拷贝的字符串长度
注解
By Wesley @ 2022.03.17 核心要点就是使用memcpy替代strcpy，原本认为对于长字符串会有性能提升
By Wesley @ 2023.10.09 重新和strcpy进行了性能测试，发现性能提升不显著，甚至有一些下降 可能与早期测试环境有关，由于用到的地方很多，目前保留不变

< 如果长度为0，自动计算源字符串长度

< 使用memcpy拷贝内存

< 手动添加字符串结束符

< 返回拷贝的字符串长度
