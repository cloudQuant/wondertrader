template<typename T>
class ObjectPool< T >

对象池模板类

模板参数
T池中管理的对象类型

该类提供对象的快速分配和回收功能，避免频繁的内存分配和释放操作， 特别适用于需要频繁创建和销毁同类型对象的场景。
