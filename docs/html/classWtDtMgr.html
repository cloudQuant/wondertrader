详细描述

数据管理器类

该类实现了IDataReaderSink和IDataManager接口 负责管理历史数据和实时行情数据，包括K线、Tick、委托队列、成交明细等 提供了数据的缓存和访问机制，是策略引擎数据访问的核心组件
