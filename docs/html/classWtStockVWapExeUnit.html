股票VWAP执行单元类

实现了基于成交量加权平均价(VWAP)策略的股票订单执行单元 根据预设的时间和数量分布，将大单拆分成多个小单，在指定时间区间内逐步执行 最大程度地降低市场冲击，提高订单执行效率
