详细描述

回测引擎类

提供了完整的策略回测环境，继承IBtDataLoader接口，实现历史数据加载 支持CTA策略、选股策略和高频策略的回测 实现了事件传递、回调调度和数据回放的核心功能 是WonderTrader回测系统与外部组件交互的核心类

void WtBtRunner::enable_tick ( bool  bEnabled = true)

启用或禁用Tick数据回测

启用或禁用tick数据回放

参数
bEnabled是否启用Tick数据，默认为true

设置回测引擎是否使用Tick数据进行回测 当启用Tick数据时，回测将基于Tick级别数据进行，精度更高，但效率较低 当禁用Tick数据时，回测将仅基于K线数据进行，效率更高，但精度较低 应在调用run方法之前调用此方法

设置是否在回测中回放造市数据 启用tick回放可以提高回测精度，但会降低性能

参数
bEnabled是否启用tick数据回放，默认为true
