相关的预定义宏:

  • NO_LOG: 关闭所有日志

  • NO_TRACE: 关闭 TRACE 级别的日志
  • NO_VA_MACRO: 不支持变参宏
  • _CHECK_LOG_FORMAT: 在编译时检查格式化字符串的格式
