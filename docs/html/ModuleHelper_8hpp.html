◆ getBinDir()

static const char * getBinDir ( )
static

获取当前模块所在的目录路径

返回
const char* 返回当前模块所在的目录路径，以斜杠结尾

该函数在不同平台下使用相应的API获取当前模块的路径 Windows平台下使用GetModuleFileName函数 Linux/Unix平台下使用getInstPath函数 然后提取路径中的目录部分，去除文件名 首次调用时计算路径并缓存，后续调用直接返回缓存值
