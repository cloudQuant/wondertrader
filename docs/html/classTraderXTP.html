撤单出错响应

参数
cancel_info撤单具体信息，包括撤单的order_cancel_xtp_id和待撤单的order_xtp_id
error_info撤单被拒绝或者发生错误时错误代码和错误信息，需要快速返回，否则会堵塞后续消息，当堵塞严重时，会触发断线，当error_info为空，或者error_info.error_id为0时，表明没有错误
session_id资金账户对应的session_id，登录时得到
备注
此响应只会在撤单发生错误时被回调

参数

报单通知
