◆ fnOnConnect

int32(* _SEndpointChannelCfg::fnOnConnect) (struct _SEndpointChannel *pAsyncChannel, void *pCallbackParams)

连接或重新连接完成后的回调函数

  • 需要通过该回调函数完成回报订阅操作
  • 若函数指针为空, 则会使用通道配置中默认的回报订阅参数进行订阅
  • 若函数指针不为空, 但未订阅回报, 90秒以后服务器端会强制断开连接
  • 若回调函数返回小于0的数, 则通信线程将中止运行

连接断开后的回调函数

  • 无需做特殊处理, 通信线程会自动尝试重建连接
