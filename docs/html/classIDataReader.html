分钟线闭合事件处理接口

当一根分钟线数据闭合时，系统调用此方法进行处理，可以触发K线数据的更新

参数
uDate闭合的分钟线日期，格式为YYYYMMDD，如20200410，这里不一定是交易日
uTime闭合的分钟线的分钟时间，格式为HHMM，如1115（表示11点15分）
endTDate如果闭合的分钟线是交易日最后一条分钟线，则endTDate为当前交易日，其他情况为0

参数

periodK线周期，如分钟线、小时线、日线等
count要读取的K线条数
etime结束时间，格式为YYYYMMDDHHMM，如果要读取到最后一条，etime设为0，默认为0
返回
K线数据片段对象指针
